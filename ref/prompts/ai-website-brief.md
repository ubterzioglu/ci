# AI Website Brief — Çi Neo Cucina

## Goal
Rebuild the existing Wix restaurant website as a modern, fast, editable website while preserving the brand story, menu content, imagery direction, and reservation/contact flow.

## Brand
Çi Neo Cucina is a chef-led restaurant in Kaş, positioned between “old” and “new” Kaş. The brand language is natural, timeless, Mediterranean, Anatolian, sustainable, calm, seasonal, and intimate.

## Core message
Doğal malzemeler, zamansız bir mutfak dili. “Eski” ve “Yeni” Kaş arasında.

## Pages to build
1. Home `/`
2. Menu `/menu`
3. About `/about`
4. Reservations `/reservations`
5. Experiences `/experiences`
6. Contact `/contact` — create as a dedicated page even though existing site uses contact section

## Content source
Use `content/pages/*.md`, `content/data/menu.json`, `content/data/contact.json`, `content/data/navigation.json`, `content/assets/image-assets.json`.

## UX priorities
Mobile-first, fast loading, clear reservation CTA, high-quality food/ambience imagery, easy-to-update menu data, SEO-friendly semantic HTML.

## Visual style
Editorial restaurant website; white/cream background, olive green, stone tones, wine accent; large imagery with whitespace; elegant serif headings.

## Must fix
Do not keep Wix footer copyright. Add clear contact/reservation CTAs. Treat wine menu as incomplete unless additional file is provided. Add address/map once provided.
